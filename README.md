## Keymaps

Leader - `SPACE`

Navigate Split Windows (Up/Down/Left/Right) - `<C-W>(j/k/h/l)`

Telescope **F**ind Files - `<Leader>f`

**R**e**s**ize Window - `<Leader>rs`

Neovim File **E**xplorer - `<Leader>e`

ToggleTerm Open/Close **T**erminal - `<Leader>t`
